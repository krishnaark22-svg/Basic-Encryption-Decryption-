🔐 Caesar Cipher Web App
A simple Encryption & Decryption Web Application built using Python (Flask) + HTML, CSS, JavaScript.
This project demonstrates the basic concept of Caesar Cipher, one of the earliest cryptographic techniques.
📌 Features
🔒 Encrypt text using Caesar Cipher
🔓 Decrypt encrypted text
🔁 Real-time API communication (Frontend ↔ Backend)
🧠 Handles uppercase, lowercase, spaces & symbols
🎨 Simple and responsive UI
🧠 How It Works
🏗️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Python (Flask)
API: REST (JSON)
⚠️ Limitations
Small key space (only 25 possible keys)
Vulnerable to brute force
Pattern preservation (frequency analysis)
⚠️ Not secure for real-world encryption
